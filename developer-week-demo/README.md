## 🎮 Interactive Controls

| Key | Action |
|-----|--------|
| `SPACE` | Pause/Resume simulation |
| `R` | Reset all particles |
| `T` | Toggle particle trails |
| `E` | Energy visualization mode |
| `M` | Toggle mouse attraction |
| `S` | Generate sound wave file |
| `Mouse Click` | Create explosion effect |
| `ESC` | Exit demo |
